One of the first tools you should would be live reloading of your web page. It lets your browser reload as changes are made! No more having to crtl+s then ctrl+f5 in your browser window. It will do it for you!! It is great for styling and layout changes.

Honestly I could have added the livereload for you pretty easily but it is more fun to do it yourself!! So when you type grunt into your command line it launches a series of tasks as specified in your Gruntfile this would be a good time to check it out and look into how it works.

There is a world of tools that you open up with a task runner like browserify, uglify, concat... you get the idea. For now lets get livereload working for you.
